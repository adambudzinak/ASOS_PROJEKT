## SPUSTENIE
Pred spustenim sa treba uistit, ze mas rozbehnuty nodejs backend (tutorial je v priecinku backend v gite)

1. npm install
2. npm run dev

## STRUKTURA PROJEKTU
- index - cast aplikacie spolocna pre vsetkych (registrovanych/prihlasenych aj neregistrovanych/neprihlasenych) <br>
- user - cast aplikacie pristupna len pre prihlasenych
- common - spolocne komponenty zdielane napriec celou aplikaciou
